Watercolor Plants in Garden Diagrams

Thirty full-color watercolor plant illustrations, installed as a shared asset library at /icons/plants/watercolor-front/ and referenced directly from SVG garden diagrams. The chat agent's system prompt now knows about them, so elevation views can be painted with real species instead of flat shapes. Four example scenes below — including a classic prairie meadow built entirely from recently-added species — and the toggle switches every scene between side profile and a bird's-eye plan view.

The Full Library (30 plants)

Every asset below is addressable from a Claude-generated diagram as /icons/plants/watercolor-front/{slug}.svg. Click any card to open the raw file.

Usage in a Claude-generated SVG

The chat agent's system prompt tells it to emit <image> tags referencing these files, sized to the suggested height, with preserveAspectRatio="xMidYMax meet" so the plant base stays anchored to the ground line.

<svg viewBox="0 0 1000 400" xmlns="http://www.w3.org/2000/svg">
  <rect y="350" width="1000" height="50" fill="#8a7a4f"/>

  <!-- Bee Balm, 155px tall, base on ground line -->
  <image href="/icons/plants/watercolor-front/bee-balm.svg"
         x="240" y="195" width="155" height="155"
         preserveAspectRatio="xMidYMax meet"/>

  <!-- A drift of Butterfly Milkweed -->
  <image href="/icons/plants/watercolor-front/butterfly-milkweed.svg"
         x="400" y="215" width="135" height="135"
         preserveAspectRatio="xMidYMax meet"/>
  <image href="/icons/plants/watercolor-front/butterfly-milkweed.svg"
         x="450" y="215" width="135" height="135"
         preserveAspectRatio="xMidYMax meet"/>
</svg>
