Key Differences from Reference

Analyzing what makes the reference icons look more realistic:

  • Opacity: Reference uses 0.4-0.7 per layer (we now match this)
  • Saturation: Reference colors are deep and rich, not washed out
  • Mottling: 30+ irregular patches of varying sizes/shades create organic depth
  • No hard outlines: The edge of each swatch fades naturally via the watercolor filter
  • Dark accents: Deep shadow patches (much darker than base) create dramatic contrast
  • Light highlights: Small bright patches on the light side add dimension
  • Green fringe: Green foliage peeks through at edges where flowers thin out
  • Ink detail: Subtle, not heavy — tiny flower marks at about 50% opacity
