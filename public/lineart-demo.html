SVG Artistic Line Art Techniques

Working examples of tapered strokes, variable line weights, dash patterns, stippling, hatching, calligraphic fills, and pencil filter effects for botanical illustration.

1. Tapered Strokes — Filled Shape Method

Two parallel bezier curves forming a closed polygon. Wide at one end, narrow at the other. Uses fill not stroke.

Basic tapered lines — thick to thin and thin to thick
Forking branch with tapered fills — each segment narrows to a point
Calligraphic strokes — S-curve, whip, and hairline taper 
<!-- Tapered line: two bezier curves converging to a point -->
<path d="M10,42 C50,38 120,36 190,49
         C120,40 50,48 10,58Z" fill="#2a3a1a"/>

2. Variable Line Weights — Structural Hierarchy

Combining thick, medium, and fine strokes creates visual depth and hierarchy. Thicker = closer/more important, thinner = detail/background.

Four weight levels: heavy outline, medium branches, fine twigs, hairline leaves

3. Stroke-Dasharray Patterns

Using stroke-dasharray with stroke-linecap="round" to create dots, dashes, and decorative line textures.

Seven dash patterns — from solid to complex decorative
Contour stippling — dots following curved paths create shading 
<!-- Dots along a contour path -->
<path d="M30,180 C40,120 60,80 100,50"
  stroke-dasharray="0 5" stroke-linecap="round" stroke-width="2"/>

4. Hatching and Cross-Hatching Patterns

SVG <pattern> elements for repeating line textures — diagonal hatching, cross-hatching, and variable density.

Pattern-based hatching — diagonal, cross, dense, and on organic shapes

5. Combined: Tapered Strokes + Watercolor + Line Art

The full botanical illustration technique: watercolor wash underneath, tapered filled branches, variable-weight ink detail on top.

Complete botanical: watercolor filter underneath, tapered filled branches, pencil-filtered leaf marks, dasharray contour stippling
Flowering perennial: tapered stems and leaves, watercolor bloom patches, daisy petal ink detail

7. Outline Pattern Variations — Thick Line Weight

Seven distinct outline patterns at 2.5px weight for defining botanical element boundaries. Each creates a different visual texture at the edge.

Sharp in/out zigzag — mimics rough leaf edges
Smooth connected arcs — classic landscape plan canopy
Irregular sharp points — rough natural edges
Right-angle steps — pixelated/architectural feel
Wobbly S-curves — hand-drawn organic feel
Nested rings with decreasing weight — contour map look
Bold dashed boundary — property lines, suggested edges

8. Tapered Outline Technique

Outlines drawn as filled shapes that vary in thickness — thick where branches push outward, thin between leaf clusters. Uses the same two-bezier filled path method as tapered strokes, but traced along the plant boundary.

Each segment is a filled shape — wide where a leaf cluster protrudes, narrow between them
Tapered outline traced on the watercolor boundary — thick lobes align with visible paint edge

9. Mixed-Technique Outlines — Switching Patterns Along One Boundary

The key insight: a single plant outline should use DIFFERENT techniques for different sections — scalloped where leaves cluster, jagged where branches poke, squiggly for soft undergrowth, boxed for structured areas. These are drawn as ONE continuous path that changes character.

Single continuous outline switching between scalloped, zigzag, boxed, jagged, and squiggly along the canopy edge
Outline switches technique based on what's underneath — scalloped at blooms, zigzag at foliage, jagged at base

  C134,40 138,48 136,54  <!-- back to scalloped -->
